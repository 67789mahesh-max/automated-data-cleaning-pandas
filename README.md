Sales Data Cleaning and Power BI Visualization
Project Overview
This project showcases an end-to-end data preparation and visualization workflow. It involves cleaning a raw, unstructured dataset using Python (Pandas) and structuring it for reporting inside a Power BI dashboard.

Technologies Used
Python (Pandas): For data ingestion, cleaning, and mathematical operations.

Jupyter Notebook: To document and execute the step-by-step data wrangling process (Pandas.ipynb).


Power BI: For building the interactive data model and reporting layout (Pandas with dashboard.pbix).
